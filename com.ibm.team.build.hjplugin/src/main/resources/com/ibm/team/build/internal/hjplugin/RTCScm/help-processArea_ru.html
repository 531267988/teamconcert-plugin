Укажите область проекта или область коллектива RTC, которая будет связана с этим заданием. Указанная область проекта или область коллектива будет использоваться для обработки потока компоновки и моментальной копии компоновки.

Для области проекта укажите имя.

Из области коллектива укажите имена всех областей коллектива в иерархии начиная с имени области проекта (имена отделяются друг от друга символом "/"). Например, Банк JKE/Разработка/Пользовательский интерфейс обозначает область коллектива Пользовательский интерфейс, которая находится в области коллектива Разработка из области проекта Банк JKE.
