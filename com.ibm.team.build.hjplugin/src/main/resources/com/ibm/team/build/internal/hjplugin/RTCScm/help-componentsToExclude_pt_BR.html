Se somente alguns componentes na área de trabalho serão construídos, será possível excluir componentes que não precisam ser carregados.

Para excluir componentes, liste-os no formato JSON a seguir, salve a lista em um arquivo no disco e especifique o caminho para o arquivo neste campo. 

  {
    "componentsToExclude":[
      {
	    "componentName":"JUnit"
      },
      {
	    "componentName":"Releng"
      }
    ]
  }

Também é possível especificar o "componentId" em vez de "componentName". O "componentId" é o ID do item (UUID) do componente. Se você especificar "componentName" e "componentId", a consulta ocorrerá somente por "componentId."
