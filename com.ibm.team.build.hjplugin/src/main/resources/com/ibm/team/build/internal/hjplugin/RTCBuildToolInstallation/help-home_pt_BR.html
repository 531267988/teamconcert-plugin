O caminho para o Kit de Ferramentas de Compilação. Este será um diretório dentro das ferramentas de compilação do RTC instalado. Exemplo: C:\Program Files (x86)\IBM\TeamConcertBuild\buildsystem\buildtoolkit

Se você pretende carregar links simbólicos, é necessário fazer uma configuração adicional. O acesso às bibliotecas nativas é necessário para a criação de links simbólicos. Para obter mais detalhes sobre a configuração, visite Team Concert Plugin
