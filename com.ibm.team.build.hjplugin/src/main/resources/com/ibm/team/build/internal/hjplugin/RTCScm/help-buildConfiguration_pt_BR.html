Definição de Compilação

A seleção de uma definição de compilação permitirá a integração com a compilação RTC. Isto permitirá que as construções sejam solicitadas a partir do RTC ou do Hudson/Jenkins. Os resultados de compilação também serão criados em ambos e vinculados um ao outro.

Com a opção Controle de Versão do Jazz, uma área de trabalho de compilação pode ser especificada juntamente com mais opções de carregamento e aceitação.


Área de Trabalho de Compilação

Selecionar uma área de trabalho de compilação apenas aceitará e buscará os conjuntos de mudanças de seus destinos de fluxo.

É possível especificar onde a área de trabalho do repositório de compilação deve ser carregada, se deve criar pastas de componentes, excluir o ambiente de simulação antes do carregamento.


Captura Instantânea da Compilação

A seleção desta opção permite carregar a captura instantânea do Jazz SCM.

É possível especificar onde a área de trabalho do repositório de compilação deve ser carregada, se deve criar pastas de componentes, excluir o ambiente de simulação antes do carregamento.


Fluxo de Compilação

A seleção de um fluxo de compilação criará uma captura instantânea e carregará a partir da captura instantânea.

É possível especificar onde a área de trabalho do repositório de compilação deve ser carregada, se deve criar pastas de componentes, excluir o ambiente de simulação antes do carregamento.
