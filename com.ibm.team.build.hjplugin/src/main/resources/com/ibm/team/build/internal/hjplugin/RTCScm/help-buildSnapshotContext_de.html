Geben Sie den Eigner der Momentaufnahme an. Der Eigner einer Momentaufnahme kann entweder ein Repository-Arbeitsbereich oder ein Datenstrom sein. Durch Angabe des Eigners der Momentaufnahme ermöglichen Sie eine bereichsorientierte Suche und vermeiden somit doppelte Werte im Suchergebnis.

Wenn Ihnen die Details zum Momentaufnahmeneigner nicht bekannt sind oder wenn sie sicher sind, dass der Momentaufnahmenname im Repository eindeutig ist, können Sie für den Momentaufnahmeneigner "- Keiner -" auswählen. Wenn Sie die UUID der Momentaufnahme angeben werden die Details zum Momentaufnahmeneigner ignoriert.

Wenn Sie "- Keiner -" für den Momentaufnahmeneigner auswählen, wird die Momentaufnahmensuche nicht auf einen Bereich begrenzt, sondern im ganzen Repository durchgeführt.

Wenn ein Repository-Arbeitsbereich oder ein Datenstrom als Eigner angegeben ist, wird der Bereich der Momentaufnahmensuche auf die Momentaufnahmen eingegrenzt, die dem angegebenen Repository-Arbeitsbereich oder Datenstrom zugeordnet sind.

Wenn als Momentaufnahmeneigener ein Repository-Arbeitsbereich ausgewählt wird, aber der Name des Repository-Arbeitsbereichs nicht angegeben wird, wird die Momentaufnahmensuche nicht auf einen Bereich eingegrenzt, sondern im ganzen Repository durchgeführt.

Wenn als Momentaufnahmeneigener ein Datenstrom ausgewählt wird, aber weder der Name des Projekt- noch des Teambereichs noch des Datenstroms, der der Eigner ist, angegeben wird, wird die Momentaufnahmensuche nicht auf einen Bereich eingegrenzt, sondern im ganzen Repository durchgeführt.

Wenn ausgewählt wird, dass der Momentaufnahmeneigner ein Datenstrom ist und wenn nur der Projekt- oder Teambereich angegeben wird, wird die Momentaufnahmensuche auf den Bereich der Momentaufnahmen begrenzt, die allen Datenströmen zugeordnet sind, deren Eigner der angegebene Projekt- oder Teambereich ist.

Wenn als Momentaufnahmeneigener ein Datenstrom ausgewählt wird und wenn nur der Datenstromname angegeben wird, wird der Bereich der Momentaufnahmensuche auf die Momentaufnahmen begrenzt, die allen Datenströmen mit dem angegebenen Namen im Repository zugeordnet sind.

Wenn ausgewählt wird, dass der Momentaufnahmeneigner ein Datenstrom ist und wenn der Name des Projekt- oder Teambereichs sowie der Namen des Eignerdatenstroms angegeben ist, wird die Momentaufnahmensuche auf den Bereich der Momentaufnahmen begrenzt, die allen Datenströmen mit dem angegebenen Namen in dem angegebenen Projekt- oder Teambereich zugeordnet sind.
