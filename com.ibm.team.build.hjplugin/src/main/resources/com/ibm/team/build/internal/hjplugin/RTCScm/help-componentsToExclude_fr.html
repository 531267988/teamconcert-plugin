Si seuls certains composants de l'espace de travail seront générés, vous pouvez exclure les composants qui n'ont pas besoin d'être chargés.

Pour exclure des composants, listez-les au format JSON suivant, sauvegardez la liste dans un fichier sur le disque et indiquez le chemin d'accès au fichier dans cette zone. 

  {
    "componentsToExclude":[
      {
	    "componentName":"JUnit"
      },
      {
	    "componentName":"Releng"
      }
    ]
  }

Vous pouvez également indiquer "componentId" au lieu de "componentName." "componentId" est l'ID élément (identificateur unique universel) du composant. Si vous indiquez à la fois "componentName" et "componentId", la recherche se produit uniquement par "componentId."
