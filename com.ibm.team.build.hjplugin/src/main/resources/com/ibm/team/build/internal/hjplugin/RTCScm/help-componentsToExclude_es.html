Si solo se van a crear algunos componentes en el espacio de trabajo, puede excluir componentes que no es necesario cargar.

Para excluir componentes, lístelos en el formato JSON siguiente, guarde la lista en un archivo en disco, y especifique la vía de acceso del archivo en este campo. 

  {
    "componentsToExclude":[
      {
	    "componentName":"JUnit"
      },
      {
	    "componentName":"Releng"
      }
    ]
  }

También puede especificar "componentId" en lugar de "componentName." El "componentId" es el ID del elemento ID (UUID) del componente. Si especifica ambos, "componentName" y "componentId", la búsqueda solo se produce por "componentId."
