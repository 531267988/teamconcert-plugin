Definizione build

La selezione di una definizione di buile consentirà l'integrazione con la build RTC. Ciò consentirà di richiedere le build da RTC o Hudson/Jenkins. Inoltre, i risultati della build verranno creati in entrambi e collegati tra loro.

Con l'opzione Jazz Source Control, è possibile specificare uno spazio di lavoro di build insieme ad ulteriori opzioni di accettazione e caricamento.


Spazio di lavoro di build

La selezione di uno spazio di lavoro di build accetterà e recupererà le serie di modifiche dalle relative destinazioni flusso.

È possibile specificare il punto in cui caricare lo spazio di lavoro del repository della build, se creare o meno le cartelle del componente, se eliminare la sandbox prima del caricamento.


Istantanea della build

La selezione di questa opzione consente di caricare un'istantanea Jazz SCM.

È possibile specificare il punto in cui caricare lo spazio di lavoro del repository della build, se creare o meno le cartelle del componente, se eliminare la sandbox prima del caricamento.


Stream della build

La selezione di uno stream della build consentirà di creare un'istantanea e di caricare dall'istantanea.

È possibile specificare il punto in cui caricare lo spazio di lavoro del repository della build, se creare o meno le cartelle del componente, se eliminare la sandbox prima del caricamento.
