Pokud budou v pracovním prostoru sestavovány pouze některé komponenty, můžete komponenty, které není třeba načítat, vyloučit.

Chcete-li vyloučit komponenty, vypište je v následujícím formátu JSON, uložte seznam do souboru na disku a určete v tomto poli cestu k souboru. 

  {
    "componentsToExclude":[
      {
	    "componentName":"JUnit"
      },
      {
	    "componentName":"Releng"
      }
    ]
  }

Rovněž můžete určit "componentId" namísto "componentName." Hodnota "componentId" je ID položky (UUID) komponenty. Určíte-li obě hodnoty "componentName" a "componentId", dojde k vyhledání pouze podle "componentId."
