Ha csak néhány komponens lesz összeépítve a munkaterületen, akkor kizárhatja a komponenseket, amelyeket nem szükséges betölteni.

Komponensek kizárásához listázza ki azokat a következő JSON formátumban, mentse a listát egy fájlba a lemezen, és adja meg a fájl útvonalát ebben a mezőben. 

  {
    "componentsToExclude":[
      {
	    "componentName":"JUnit"
      },
      {
	    "componentName":"Releng"
      }
    ]
  }

A "componentName" helyett megadhatja a "componentId" tulajdonságot is. A "componentId" a komponens elemazonosítója (UUID). Ha a "componentName" és a "componentId" egyaránt meg van adva, akkor a kikeresés csak a "componentId" alapján történik.
