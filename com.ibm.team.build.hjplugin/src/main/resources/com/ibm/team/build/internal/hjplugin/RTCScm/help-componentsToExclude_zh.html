如果将仅构建工作空间中的某些组件，那么可以排除不需要装入的组件。

要排除组件，请按以下 JSON 格式列示它们，将该列表保存到磁盘上的文件，并在此字段中指定该文件的路径。 

  {
    "componentsToExclude":[
      {
	    "componentName":"JUnit"
      },
      {
	    "componentName":"Releng"
      }
    ]
  }

您还可以指定“componentId”，而不指定“componentName”。“componentId”是组件的项标识 (UUID)。如果您同时指定“componentName”和“componentId”，那么只能通过“componentId”进行查找。
