Если будут скомпонованы только некоторые компоненты в рабочей области, можно исключить те компоненты, которые не требуется загружать.

Для исключения компонентов создайте их список в следующем формате JSON, сохраните список в файле на диске и укажите путь к файлу в данном поле. 

  {
    "componentsToExclude":[
      {
	    "componentName":"JUnit"
      },
      {
	    "componentName":"Releng"
      }
    ]
  }

Вместо "componentName" можно указать "componentId". "componentId" задает присвоенный компоненту ИД элемента (UUID). Если задано как "componentName", так и "componentId", используется только "componentId".
