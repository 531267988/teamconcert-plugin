Se verranno creati solo alcuni componenti dello spazio di lavoro è possibile escludere i componenti che non devono essere caricati.

Per escludere dei componenti, elencarli nel seguente formato JSON, salvare l'elenco in un file su disco e specificare il percorso del file in questo campo. 

  {
    "componentsToExclude":[
      {
	    "componentName":"JUnit"
      },
      {
	    "componentName":"Releng"
      }
    ]
  }

È anche possibile specificare "componentId" invece di "componentName." "componentId" è l'ID elemento (UUID) del componente. Se "componentName" e "componentId" vengono entrambi specificati, la ricerca viene eseguita solo per "componentId."
