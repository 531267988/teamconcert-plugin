Specify the RTC project area or team area that will be be used to resolve the stream that owns the snapshot. When a project or team area is specified, without the associated stream, the snapshot search is scoped to all streams owned by the specified project or team area.

For project area, specify the name.

For team area, specify the name of all team areas in the hierarchy, starting with the name of the project area, with each of the names separated by "/". For an instance, JKE Banking/Development/User Interface identifies the 'User Interface' team area which is under the 'Development' team area in the 'JKE Banking' project area.
