Укажите область проекта или коллектива RTC, в которой будет выполняться поиск настроенного в задании потока.

Для области проекта укажите имя.

Для области коллектива укажите имена всех областей коллектива в иерархии начиная с имени области проекта (имена отделяются друг от друга символом "/"). Например, Банк JKE/Разработка/Пользовательский интерфейс обозначает область коллектива Пользовательский интерфейс, которая находится в области коллектива Разработка из области проекта Банк JKE.
