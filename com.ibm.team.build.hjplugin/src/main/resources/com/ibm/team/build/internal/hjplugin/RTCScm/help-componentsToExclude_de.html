Wenn nur manche Komponenten im Arbeitsbereich erstellt werden sollen, können Sie Komponenten ausschließen, die nicht geladen werden müssen.

Um Komponenten auszuschließen, listen Sie sie im folgenden JSON-Format auf, speichern Sie die Liste in einer Datei auf einem Datenträger und geben Sie den Pfad zur Datei in diesem Feld an. 

  {
    "componentsToExclude":[
      {
	    "componentName":"JUnit"
      },
      {
	    "componentName":"Releng"
      }
    ]
  }

Sie können auch "componentId" (Komponenten-ID) anstelle von "componentName" (Komponentenname) angeben. Bei "componentId" handelt es sich um die Element-ID (UUID) der Komponente. Wenn Sie sowohl "componentName" als auch "componentId" angeben, erfolgt die Suche nur für "componentId."
