Build Definition

Selecting a build definition will allow integration with RTC build. It will allow builds to be requested from either RTC or Hudson/Jenkins. Build results will also be created in both and linked to each other.

With the Jazz Source Control option, a repository workspace can be specified along with more load and accept options.


Repository Workspace

Selecting a repository workspace will just accept and fetch the change sets from its flow target(s).

You can specify where the repository workspace should be loaded, whether to create component folders, delete the sandbox prior to loading.


Snapshot

Selecting this option allows you to load a Jazz SCM snapshot.

You can specify where the repository workspace should be loaded, whether to create component folders, delete the sandbox prior to loading.


Stream

Selecting a stream will create a snapshot and load from the snapshot.

You can specify where the repository workspace should be loaded, whether to create component folders, delete the sandbox prior to loading.
