Pfad zum Build Toolkit. Dies ist ein Verzeichnis in der Installation der RTC Build Tools. Beispiel: C:\Program Files (x86)\IBM\TeamConcertBuild\buildsystem\buildtoolkit

Wenn Sie beabsichtigen, symbolische Links zu laden, müssen zusätzliche Konfigurationsschritte ausgeführt werden. Um symbolische Links zu erstellen, ist ein Zugriff auf native Bibliotheken erforderlich. Weitere Informationen zur Konfiguration finden Sie unter Team Concert Plugin.
