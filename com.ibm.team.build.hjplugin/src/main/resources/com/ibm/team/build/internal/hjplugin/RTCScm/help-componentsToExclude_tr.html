Çalışma alanındaki yalnızca bazı bileşenler oluşturulacaksa, yüklenmesi gerekmeyen bileşenleri dışlayabilirsiniz.

Bileşenleri dışlamak için aşağıdaki JSON biçiminde bunları listeleyin, listeyi diskte bir dosyaya kaydedin ve bu alanda dosyanın yolunu belirtin. 

  {
    "componentsToExclude":[
      {
	    "componentName":"JUnit"
      },
      {
	    "componentName":"Releng"
      }
    ]
  }

"componentName" yerine "componentId" belirtebilirsiniz. "componentId", bileşenin öğe tanıtıcısıdır (UUID). Hem "componentName" hem "componentId" belirtirseniz, arama yalnızca "componentId" değerine göre yapılır.
