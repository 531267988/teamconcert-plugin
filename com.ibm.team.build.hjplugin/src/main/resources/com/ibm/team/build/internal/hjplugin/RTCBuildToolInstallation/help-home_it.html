Il percorso del toolkit della build. Questa sarà una directory compresa nei tool di build RTC installati. Esempio: C:\Program Files (x86)\IBM\TeamConcertBuild\buildsystem\buildtoolkit

Se si intende caricare link simbolici, è necessario eseguire alcune configurazioni aggiuntive. L'accesso alle librerie native è richiesto per creare link simbolici. Per ulteriori dettagli sulla configurazione, visitare Team Concert Plugin
