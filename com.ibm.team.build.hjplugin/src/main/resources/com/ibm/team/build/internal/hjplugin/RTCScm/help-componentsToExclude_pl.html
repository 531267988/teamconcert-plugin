Jeśli w środowisku roboczym zostaną zbudowane tylko niektóre komponenty, można wykluczyć te, których nie trzeba ładować.

Aby wykluczyć komponenty, należy utworzyć ich listę w następującym formacie JSON, zapisać tę listę w pliku na dysku i podać w tym miejscu ścieżkę do tego pliku. 

  {
    "componentsToExclude":[
      {
	    "componentName":"JUnit"
      },
      {
	    "componentName":"Releng"
      }
    ]
  }

Element componentName można zastąpić elementem componentId. Element componentId jest identyfikatorem UUID komponentu. Jeśli zostaną określone oba elementy - componentName i componentId, komponent zostanie wyszukany tylko według elementu componentId.
